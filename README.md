# 50 Projects in 50 Days 💻

50 mini web projects built with vanilla HTML, CSS & JavaScript — no frameworks, no build tools. Each project explores a specific concept or UI pattern.

## Key Learnings 📚

- Webpage layout and structuring with HTML
- CSS styling, Flexbox, Grid, transitions, keyframes, transforms
- Responsive design
- DOM manipulation and JavaScript events
- Fetching data from third-party APIs using Fetch
- Array methods — `map()`, `filter()`, `reduce()`
- JSON, Promises, Async/Await, localStorage, Canvas API
